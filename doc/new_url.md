# New URL

Create a new URL.

**URL** : `/urls`

**Method** : `POST`

**Auth required** : YES

**Permissions required** : None

## Success Response

**Code** : `200 OK`

**Response examples**

```json
{
    "id": 1,
    "original_url": "https://www.hi.com",
    "slug": "4a5ohzgv",
    "active": true,
    "user_id": 1,
    "created_at": "2021-02-23T22:36:36.607Z",
    "updated_at": "2021-02-23T22:36:36.607Z"
}
```

## Notes

*  Body of the request should be JSON.
